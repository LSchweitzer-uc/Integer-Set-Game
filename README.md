# Integer-Set-Game
Integer Set based on the talk: "Projective and Non-Abelian Set" by Catherine Hsu

The concept is based on the abstraction of the popular game Set to a general group. The usual game of Set has the structure of the group (Z/3Z)^4. However the concept of finding a set of elements that sums to 0 is easily generalized to many Sets, even non-abelian ones. 
The integers are abelian, however they are infinite, so a physical game is not possible. Most integers are also far too large to think about and even for a computer to store. To fix this the integers in this game are normally distributed about 0 (so technically any integer could be part of your game but the likelihood of the game working is still large).
