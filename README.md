# Integer-Set-Game
Integer Set based on the talk: "Projective and Non-Abelian Set" by Catherine Hsu

We can't actually choose any random integer due to lack of storage space on the computer, so I've settled on a standard deviation around 0.
Highscore is the highest score reached in a timed game multiplied by the difficulty of that game (might update this formula)
difficulty is set to the new input value when start game is pressed
